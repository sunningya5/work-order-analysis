# AI质检工单准确率分析系统

## 一、项目概述

基于物流工单质检规则，对AI质检结果进行准确率分析。系统从工单数据中提取质检信息，按规则判定有责/无责，与人工复核结果对比，计算各规则的准确率。

数据来源：工作簿4（规则定义）、工作簿5（人工复核测试反馈）、工作簿6（规则分类）。

### 工作簿5.xlsx — 人工复核测试反馈（50条）

字段：序号、反馈时间、运单号、工单编码、结办日期、反馈人、工单池类型、问题描述、优化建议、IT回复、总结

问题描述中记录了每条规则的AI判定是否正确，用于计算准确率。

### 工作簿6.xlsx — 质检规则分类（17条在线规则）

字段：序号、质检类型（AI输出）、质检子类(参考）、质检说明（AI输出）

为工作簿4的精简版，仅保留在线规则。

## 四、核心指标

- **首检正确率** = AI有责数 / 首检有责规则数
- **复检正确率** = AI有责数 / 复检有责规则数
- **AI有责规则数**：该工单内AI判定为有责的规则数量

## 五、报表功能

### 筛选条件

- 首次质检时间（起/止）
- 结办时间（起/止）
- 工单大类
- 工单小类
- AI质检总结果（有责/无责）
- 是否复检

### KPI看板

- 总工单数
- AI判定有责/无责
- 已复检数
- 首检正确率（当日/当月，可点击查看趋势）
- 复检正确率（当日/当月，可点击查看趋势）

### 质检类型准确率

- 4张类型卡片，显示准确率、正确/总数、规则标签
- 点击卡片 → 趋势折线图（日度/月度切换，数据标签直接显示）
- 右上角「规则明细」按钮 → 展开该类型下所有规则详情

### 工单明细表格

- 22个字段完整展示
- 分页浏览（20/50/100条/页）
- 导出Excel

## 六、技术栈

- Python 3 + pandas / openpyxl
- Chart.js 4（折线图，jsdelivr CDN）
- 纯HTML/CSS/JS，零框架依赖

## 七、使用方式

```bash
# 安装依赖
pip install pandas openpyxl numpy

# 生成报表（使用示例数据）
python build_8089.py

# 接入真实数据
python build_8089.py 正式工单数据.xlsx
```

生成 `AI质检工单准确率明细表_8089.html`，浏览器直接打开即可。
