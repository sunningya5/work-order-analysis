# 工单质检报表数据分析系统

基于AI质检规则的工单准确率分析平台，支持质检类型下钻、趋势图、筛选导出等功能。

## 快速开始

### 安装依赖

```bash
pip install pandas openpyxl numpy
```

### 生成报表

```bash
# 8088 报表（侧边栏筛选 + 图表）
python generate_detail_report.py

# 8089 报表（顶部筛选 + 卡片式）
python build_8089.py
```

### 接入真实数据

```bash
python generate_detail_report.py 正式数据.xlsx
```

脚本会自动匹配字段名，计算首检正确率和复检正确率。

## 数据文件

| 文件 | 说明 |
|------|------|
| 工作簿4.xlsx | 质检规则定义（20条，含原始逻辑、规则细化、AI判断逻辑） |
| 工作簿5.xlsx | 人工复核测试反馈（50条） |
| 工作簿6.xlsx | 质检规则分类（17条在线规则） |

## 指标说明

- **首检正确率** = AI有责数 / 首检有责规则数
- **复检正确率** = AI有责数 / 复检有责规则数
- **质检类型**：虚假回访、随意建单、虚假处理、登记不规范

## 报表功能

### 8089 卡片式报表

- 顶部筛选栏（日期、工单大类/小类、AI结果、是否复检）
- KPI卡片（总工单数、有责/无责、已复检、当日/当月正确率）
- 质检类型卡片（点击 → 趋势折线图，右上角规则明细按钮）
- 工单明细表格（分页、导出Excel）
- 趋势图数据标签直接显示

### 8088 经典报表

- 左侧筛选栏
- 质检类型柱状图（点击下钻到规则明细 + 日度/月度趋势）
- 首检/复检正确率对比图
- 工单明细表格（分页、导出CSV）


## 部署

服务器：Ubuntu 22.04 + Python HTTP Server + Nginx

```bash
# 上传文件
scp AI质检工单准确率明细表_8089.html ubuntu@服务器IP:/home/ubuntu/reports2/index.html

# 重启服务
ssh ubuntu@服务器IP "pkill -f server8089; cd /home/ubuntu/reports2 && nohup python3 /home/ubuntu/server8089.py &"
```
